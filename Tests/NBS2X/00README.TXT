This folder contains BASIC programs to test various features of bwBASIC.


The E###.* files implement automated tests of various extensions to ECMA-55.
The E###.* files have been written for bwBASIC by Howard Wulf, AF5NE.


The M###.* files implement manual tests of various extensions to ECMA-55.
The M###.* files have been written for bwBASIC by Howard Wulf, AF5NE.

The F_*.* files implement automated stress tests of various instrinsic functions.
